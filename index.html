Description

This course introduces the theory and design of programming languages, including syntax, semantics, type systems, and program reasoning. Topics may include lambda calculus, operational semantics, type safety, and language implementation techniques.

Prerequisites

Data structures, discrete mathematics, and some programming experience.

Textbooks and Readings

  • Benjamin C. Pierce, Types and Programming Languages
  • Michael Sipser, Introduction to the Theory of Computation (selected topics)
  • John S. McCarthy, website
